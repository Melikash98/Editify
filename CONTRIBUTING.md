## About Editify

**Editify** is a customizable Android UI component library focused on creating modern input fields with:

- Floating animated hints  
- Helper / Warning / Error states  
- Password visibility toggle  
- RTL / LTR support  
- XML customization  
- Custom fonts and colors  
- Material-inspired UI behavior  

---

## Ways to Contribute

You can help by contributing in these areas:

### 🐛 Bug Reports

If you found a bug:

- Explain the issue clearly
- Include Android version
- Include device model
- Include library version
- Add screenshots or screen recording if useful
- Include reproduction steps

Use GitHub Issues.

---

### 💡 Feature Requests

Suggestions are welcome, especially for:

- New input states
- More animations
- Better accessibility support
- Compose compatibility
- Performance improvements
- Additional XML attributes
